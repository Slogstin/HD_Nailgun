# HD Nailgun

This Addon adds Nailgun and steel nails to [Hideous Destructor](https://codeberg.org/mc776/HideousDestructor).

Requires [HDCoreLib](https://github.com/HDest-Community/hdest-core-lib).

## Contents

### VIRID Inc. Electro magnetic Nailgun

    * Can spawn on Chainguns
    * Uses Steel Nails and batteries
    * Use Altfire to fire faster!
    * Altfire drains the battery twice as fast
    * A bit front heavy when mag has a lot of nails
    * Fits in a backpack
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: VNG
    * Class: VirNailgun

### Nailgun Drum Magazine

    * Can spawn on Clip Boxes
    * Holds 35 Nails
    * Quite bulky when full
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: VNM
    * Class: VirNailgunMag

### Steel Nails

    * Can spawn on Cell Packs and Shell boxes
    * A box holds 70 Nails
    * Nails can bend when they hit a wall
    * Bent nails can be picked up, but not used
    * Can be sold by Merchants
    * Price and Spawn weight can be adjusted in the menu
    * Loadout code: VNL
    * Class: VirNailAmmo

### Nail Repair Device

    * Only spawns in Backpacks
    * Can repair bent steel nails
    * It doesn't explode, but repairing takes a long time
    * Can be sold by Merchants
    * Price can be adjusted in the menu
    * Loadout code: VNR
    * Class: NailReloader
